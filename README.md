Spotify Clone 🎧

This is a Spotify UI Clone built using HTML and CSS.
The project replicates the basic layout and design of the Spotify web player.

🚀 Features

- Spotify-inspired user interface
- Sidebar navigation (Home, Search, Library)
- Featured charts section
- Music player layout
- Responsive design

🛠 Technologies Used

- HTML5
- CSS3

📂 Project Structure
